Задача 2.
Напишите программу арифметического сообщения заданного ряда. використовуйте операции +,-,/,*. приоритет операций стандарт.

Пример:

2+2 => 4;

1+2*3 => 7;

1-2*3 => -5;

использовать возможность использования скобок, меняющих приоритет операций.
Пример:

1+2*3 => 7;

(1+2)*3 => 9;
